# Ai_Voice_Assistant
Combines multiple Ai models into a voice assistant using python.

Currently supported models (as of Feb 5, 2025):

Gemini-1.5-Flash



Models that are currently not supported but will be supported in the near future:

ChatGPT-4o
Gemini 2.0
Deepseek V3
Deepseek R1
Anthropic
