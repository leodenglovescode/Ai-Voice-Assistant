# Ai_Voice_Assistant
This is a program that combines multiple LLMs into a voice assistant using python.<br><br> 

Currently supported models (as of Feb 5, 2025):<br> 
Gemini-1.5-Flash ✅<br> 
ChatGPT-4o ❌<br> 
Gemini 2.0 ❌<br> 
Deepseek V3 ❌<br> 
Deepseek R1 ❌<br> 
Anthropic ❌<br>
